# MobAi Warehouse Backend

This is the backend service for the MobAi Warehouse Management System.

## Getting Started

### Prerequisites
- Java 17+
- Maven 3.6+
- PostgreSQL (or H2 for testing)
/
### Running the Application

```bash
mvn spring-boot:run
```
